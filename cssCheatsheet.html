CSS Cheatsheet

  • *: * is a selector element used to apply the styling universally
  • #: #xyz can be used to select any element having a particular id.
  • width: used to set width of an element.
  • height: used to set height of an element
  • color: used to set the color of the text
  • text-align: used to align text within the parent element
  • background-color used to set background color of the element
  • border: used to set border of around an element
  • border-radius: used to make the corner of the border round.
  • margin: used to give margin around the element, i.e. to leave empty space around the element
  • padding: used to give padding to an element, i.e. to leave empty space around the content of the element and within the border.
  • position: used to set position of the specific element
    Values of this property are fixed, absolute, relative, sticky, etc.
  • display: used to set different type of displays to an element to boost visuals
    Values of this property are inline, block, inline-block, flex, flexbox, etc.
  • flex-direction: used to set the direction of flow of elements in a flex [rows(for vertical flow) / columns(for horizontal flow)]
  • justify-content: used to align elements in a flex vertically or horizontally depending on the flex-direction.
  • align-items: used to align elements on the opposite axis to that of flex-direction.
  • size-units: different size units are used to represent the size of an element or text, some are relative and some are not. Some examples are px, em, rem, vh, vw, etc
  • font-size: used to set the size of the font
  • font-weight: used to set the boldness of the text/font
